## 4. 快速定位步骤

1. 打开 screenshot，确认失败发生在哪一步。
2. 使用 `npx playwright show-trace <trace.zip>` 对照 action 与 network 时间线。
3. 检查 `request-log.json`：
   - 是否有关键请求未发出。
   - 是否命中错误 URL 或错误方法。
   - 是否出现 4xx/5xx 或 `requestfailed`。
4. 若是断言问题，核对页面文案/选择器是否发生变更。
5. 若是接口问题，回放 mock 或后端返回并复现实例。
